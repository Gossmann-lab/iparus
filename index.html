IPARUS: Genomic Resources for the Parus genus

Integrated Platform for Parus Alignments, Resources and Useful Sequences

Providing public access to genomic data of the tit family.

Bioinformatic Pipeline

Trimmed reads were aligned to the Parus major reference genome (GCF_001522545.3) using BWA-MEM2. Processed with GATK v4.5.0 for high-accuracy variant calling.
